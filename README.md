# 🏺 Espaço-Cerâmica

Projeto desenvolvido para a disciplina de **Web Front-End**, consistindo em um site institucional e catálogo virtual de uma loja de cerâmica artesanal.

## 📖 Sobre o Projeto

O Espaço-Cerâmica foi criado para apresentar produtos artesanais em cerâmica de forma moderna, elegante e responsiva.

O site permite aos usuários:

- Conhecer a coleção de produtos
- Visualizar informações da empresa
- Adicionar produtos ao carrinho
- Simular uma compra utilizando pagamento via Pix
- Entrar em contato através do formulário de atendimento

---

## ✨ Funcionalidades

### 🛒 Carrinho de Compras
- Adição de produtos ao carrinho
- Atualização automática da quantidade de itens
- Cálculo automático do valor total da compra

### 💳 Pagamento Pix
- Simulação de pagamento via Pix
- Exibição de QR Code ilustrativo
- Botão para copiar o código Pix

### 📱 Layout Responsivo
- Compatível com computadores
- Compatível com tablets
- Compatível com smartphones

### ♿ Acessibilidade
Foram implementados recursos de acessibilidade utilizando JavaScript:

- Aumento do tamanho da fonte (A+)
- Redução do tamanho da fonte (A-)

Esses recursos auxiliam usuários com dificuldades visuais durante a navegação.
